# Resume 插件

> 简历相关技能集合 —— 从 Git 仓库提取项目经验、生成简历项目描述。

## 技能列表

| 技能 | 说明 | 触发方式 |
|------|------|---------|
| `extract-project` | 从 Git 仓库提取个人项目经验，生成符合 STAR 法则的简历文档 | 提供仓库地址 + "提取项目经验" |

## 安装

```bash
# 本地路径安装
/plugin install ./plugins/resume

# 或通过 Marketplace（发布后）
/plugin install resume@poseidon
```
