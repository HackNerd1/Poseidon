# Worker Contract：行为面试故事库

## 输入

- 简历中工作经历的 SARL bullet
- JD 中要求的软技能关键词（如有）

## 输出

写入 `interview/behavioral/stories.md`

## 模板

```markdown
# 行为面试故事库

## 故事 N：<标题>

**关联 bullet**：<对应的简历 bullet>

**S**：<背景> | **T**：<任务> | **A**：<方案> | **R**：<成果> | **L**：<沉淀>

**适用问题**：
- "<常见行为面试问题>"
- ...

**JD 关联**（如有 JD）：
- <此故事可应对 JD 中的哪些软技能要求>
```

## 规则

- 每个 SARL bullet 提炼一个 STARL 故事卡
- 适用问题列出 2-4 个常见行为面试问题
- 有 JD 时，标注每个故事可覆盖 JD 中的哪些软技能/素质要求
